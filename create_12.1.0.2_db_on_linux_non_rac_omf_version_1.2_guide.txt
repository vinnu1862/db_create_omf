Objective: Create Oracle Database (non-RAC, using Oracle Managed Files) on Linux using Database Template
Date of last update: 10-May-2018
Version: 1.2

------------------------------------------------------------------------------------------------------------------
1. Create working directory

$ cd /u03/work
$ mkdir <work_directory>

------------------------------------------------------------------------------------------------------------------
2. Copy the following 2 files into working directory created in Step 1

create_12.1.0.2_db_on_linux_non_rac_omf_version_1.2_template.dbt
create_12.1.0.2_db_on_linux_non_rac_omf_version_1.2_script.sh
create_profiles_in_12.1.0.2_database.sql
DBCheckList.sh

------------------------------------------------------------------------------------------------------------------
3. Update the script to create the database

$ cd /u03/work/<work_directory>
$ vi create_12.1.0.2_db_on_linux_non_rac_omf_version_1.1_script.sh

Review the "Start of Configuration Section" and update the following lines:

dbname=XXXXXXX            # specify database name in lower case
sysPassword=XXXXXXX       # specify password generated by "pwgen -y" utility
systemPassword=XXXXXXX    # specify password generated by "pwgen -y" utility
dbsnmpPassword=XXXXXXX    # specify password generated by "pwgen -y" utility

Replace the "XXXXXXX" strings appropriately

------------------------------------------------------------------------------------------------------------------
4. Change Linux permission on the script to make it executable

$ cd /u03/work/<work_directory>
$ chmod u+x create_12.1.0.2_db_on_linux_non_rac_omf_version_1.2_script.sh

------------------------------------------------------------------------------------------------------------------
5. Create database in silent mode by running the script using the template

$ cd /u03/work/<work_directory>
$ ./create_12.1.0.2_db_on_linux_non_rac_omf_version_1.2_script.sh

The script will start creating database and will show the progress of percentage completed.

Typicaly it takes about 17 minutes for this script to create the database.

------------------------------------------------------------------------------------------------------------------
6. Update the Bash Profile to include alias for the newly created database

$ cd ~
$ vi .bash_profile

add the following type of line into the .bash_profile file (replace "dbname" by the proper database name)

alias dbname='export ORACLE_SID=dbname;export ORACLE_HOME=$ORACLE_HOME;export PATH=$ORACLE_HOME/bin:/bin:/usr/bin'

$ . ./.bash_profile

------------------------------------------------------------------------------------------------------------------
7. Create profiles by running the "create_profiles_in_12.1.0.2_database.sql" script

$ dbname  (replace "dbname" by the proper database name)

Verify if enviroment was set properly:

$ env | grep ORA

$ sqlplus / as sysdba

SQL> @create_profiles_in_12.1.0.2_database.sql

------------------------------------------------------------------------------------------------------------------
8. Complete post-database creation verification

Either use the automatic script DBCheckList.sh to perform the verification:

$ cd /u03/work/<work_directory>
$ chmod u+x DBCheckList.sh
$ ./DBCheckList.sh <db_name> nonrac

or follow the manual steps per the "create_12.1.0.2_db_on_linux_non_rac_omf_version_1.2_verification.txt" document

------------------------------------------------------------------------------------------------------------------
9. Perform database discovery in OEM as per instructions in "Database_and_Related_Targets_Discovery_in_OEM.docx" file.
